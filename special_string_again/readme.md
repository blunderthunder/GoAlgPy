</style><svg style="display: none;"><defs id="MathJax_SVG_glyphs"></defs></svg><p>A string is said to be a <em>special string</em> if either of two conditions is met:  </p>

<ul>
<li>All of the characters are the same, e.g. <code>aaa</code>.</li>
<li>All characters except the middle one are the same, e.g. <code>aadaa</code>.</li>
</ul>

<p>A <em>special substring</em> is any substring of a string which meets one of those criteria.  Given a string, determine how many special substrings can be formed from it.  </p>
